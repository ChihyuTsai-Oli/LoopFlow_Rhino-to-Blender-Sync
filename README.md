# LoopFlow — Rhino to Blender Sync

> **Embrace the loop. Let it flow.**

Bridge Rhino 3D models directly into Blender — keeping geometry, materials, and scene structure in sync without the manual export-import cycle.

[▶ Watch on YouTube](https://www.youtube.com/@LoopFlow) · [📦 Releases](https://github.com/ChihyuTsai-Oli/LoopFlow_Rhino-to-Blender-Sync/releases) · [📋 Changelog](memo.md)

## Features

- **Model Sync** — One-click export from Rhino; Blender updates the mesh while preserving existing materials
- **Camera Sync** — Mirrors the active Rhino viewport to the Blender camera in real time
- **Light Alignment** — Rhino Point objects drive automatic light fixture placement in Blender
