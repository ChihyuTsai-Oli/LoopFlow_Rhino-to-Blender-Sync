# LoopFlow — Rhino 到 Blender 同步

> **擁抱循環，讓它流動。**

將 Rhino 3D 模型直接橋接到 Blender，保持幾何體、材質與場景結構的同步，省去手動匯出匯入的流程。

[▶ 觀看示範（YouTube）](https://www.youtube.com/@LoopFlow) · [📦 Releases](https://github.com/ChihyuTsai-Oli/LoopFlow_Rhino-to-Blender-Sync/releases) · [📋 更新日誌](memo.md)

## 功能

- **模型同步** — 一鍵從 Rhino 匯出；Blender 更新幾何體並保留既有材質
- **相機同步** — 即時將 Rhino 作業視窗的視角鏡射到 Blender 相機
- **燈光對齊** — Rhino 點物件自動驅動 Blender 的燈具位置對齊
